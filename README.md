# SwcosClientMap
Client Map for SWCOS via Angular.

**Setup**
  * `npm install`
  * `bower install`
  * `grunt serve`

**Additional bower modules**
  * `bower install angular-google-maps --save`
  * `bower install angular-chartist --save`
  * `bower install angular-ui-router --save`
  * `add miseed ones that get thrown`

**Port**
  * `localhost:8000`

**EndPoint on Server**
  * get "/api/uploads"
